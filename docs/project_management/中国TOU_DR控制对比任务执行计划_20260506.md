## 1. 主线目标

主线研究问题固定为两组：

1. 中国分时电价与尖峰电价条件下，TES 是否提高经济型 MPC 收益，收益如何随电价结构变化。
2. DR 事件或 peak-cap 约束下，TES 是否提高削峰能力，同时不恶化温度约束与求解可行性。

控制器固定为：

- `direct_no_tes`：在代码中对应 `no_tes`。
- `rbc_tes`：在代码中对应 `rbc`。
- `mpc_no_tes`：同一 MILP 禁用 TES。
- `mpc_tes`：在代码中对应 `mpc`。

主线不新增复杂 AI 控制层；灰盒辨识、残差修正和 RL-lite 只作为扩展。

- solver time 的均值、P95 和最大值。

- peak-cap 或 DR 事件削减是否达成。

## 7. 正式矩阵结果摘要

- 正式结果目录：`results/china_tou_dr_matrices_20260506/`。
- 完成情况：138 / 138 runs；每个正式 run 为 30 天、2880 steps。
- 可行性：`fallback_count` 总计 1；最低 `feasible_rate = 1.0`；最低 `optimal_rate = 0.999653`。
- 计算时间：最大 `solve_time_p95_s = 3.7692 s`。
- `mpc_no_tes` 到 `mpc` 的 paired 月成本差：平均节省 518.81 CNY，bootstrap CI 为 313.84 到 737.38 CNY，`n_pairs=61`。
- TOU screening 中，`mpc` 平均月成本为 1,291,919.83 CNY，`mpc_no_tes` 为 1,292,366.36 CNY；paired 平均节省 446.52 CNY。
- Peak-cap 中，`r_cap = 1.00, 0.99, 0.97, 0.95` 对应达成峰值约为 21090.0、20879.1、20457.3、20035.5 kW，`peak_slack_max_kw` 近似为 0。
- DR event 中，每个场景只触发 1 次事件；`r_DR = 0.05, 0.10, 0.15` 的请求削减为 1900、3800、5700 kWh，当前代理模型达成削减为 2060.78 kWh。
- 结果边界：最严格 `r_cap=0.95` peak-cap 场景出现约 415-431 degree-hours 温度违约，必须作为压力测试讨论；DR 收益仍是情景估算。
