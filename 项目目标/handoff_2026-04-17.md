## 8. 技术细节备忘

### EnergyPlus 不支持 loop pump + branch pump 共存
- E+ 23.1 限制：一个 PlantLoop 内要么全用 loop pump（放在 Inlet Branch），要么全用 branch pump
- 我们的 CHW Loop 有 loop pump（Chilled Water Loop Secondary Pump），所以不能给 TES 加 branch pump
- 替代方案：用 EMS System Node Setpoint 直接控制 TES 节点的 Mass Flow Rate Max/Min
- 这是一个 workaround，不是 E+ 推荐做法。如果流量控制不精确，考虑切换到全 branch pump 架构（需要给 Chiller 分支也加泵）

### Wrapper 链顺序
```
SB3 → Monitor(auto) → DummyVecEnv(auto) → LoggerWrapper → NormalizeObservation → TESIncrementalWrapper → EplusEnv
```
- TES wrapper 在 NormalizeObservation 之前，所以 valve_position 也会被归一化
- valve_position 范围 [-1, 1]，归一化后均值约 0、方差约 0.33

### SOC 计算公式（EMS P_6）
```
T_tank_avg = mean(T_node_1 ... T_node_10)
SOC = (12.0 - T_tank_avg) / 6.0    // 6°C=满冷(SOC=1), 12°C=空罐(SOC=0)
SOC = clip(SOC, 0, 1)
```

### 增量控制参数
- δmax = 0.20
- 从 idle 到满充/放: 1.0 / 0.20 = 5 步 = 5 小时
- 从满充到满放: 2.0 / 0.20 = 10 步 = 10 小时
